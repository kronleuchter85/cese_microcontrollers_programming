Para ejecutar este programa es necesario:

1) Compilar y flashear en la placa nucleo
2) Configurar una TTy o emulador de terminal serie con los siguientes parametros:
   - paridad: Odd (impar)
   - stop bits: 1
   - data bits: 7 bits (porque se usa 1 de parada)
   - baud rate: 9600 baudios
   - COM port: 3
3) una vez conectado presionar 'a' o 'b' y probar el funcionamiento. Tener en cuenta que se desarrollo en Windows por lo que el retorno de carro esta implementado con los caracteres para dicho sistema operativo.
